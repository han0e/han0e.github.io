Salarium에 오신 것을 환영합니다!

Salarium이란?

복잡하고 번거로운 급여 계산 및 관리 업무를 간소화하는 스마트 급여 관리 프로그램입니다.

기대 효과

단순 반복 업무에서 벗어나 더 중요한 핵심 업무에 집중할 수 있도록 돕습니다.
정확하고 빠른 급여 처리로 업무 효율을 극대화하세요.

설치 및 시작하기

시스템 요구 사항

  • 운영체제: Windows 10 이상
  • .NET 런타임: .NET 8.0 이상

설치 방법

  1. Salarium-Setup-vX.X.X.exe 파일 실행
  2. 화면 안내에 따라 설치 진행
  3. 바탕화면 아이콘으로 프로그램 시작

간편한 직원 정보 관리

보험료 고지내역 업로드

사회보험통합징수포털에서 받은 건강보험료, 연금보험료 파일을 업로드하여 직원별로 적용합니다.

처리 순서

건강보험료, 연금보험료 csv파일 업로드 후 우측 하단 적용하고 직원관리로 이동 버튼을 클릭합니다.

간편한 직원 정보 관리

직원 관리

각 직원별로 급여 및 혜택내역, 기간을 관리하는 페이지입니다.

처리 순서

1. 급여계산을 실행 할 직원을 선택하여 우측상단 파란색 선택직원 급여계산버튼을 클릭합니다.

기능

1. 직원관리 페이지 엑셀 내보내기 : 현재 페이지의 내용을 엑셀로 내보냅니다.

2. 직원관리 페이지 엑셀 불러오기 : 편집 후 엑셀 파일을 불러옵니다.

3. 직원관리 페이지 DB 저장 : 현재 페이지의 내용을 DB에 저장합니다.(되돌리기 불가능)

4. 직원관리 페이지 DB 불러오기 : DB에 저장된 내용을 불러옵니다.

5. 정렬 및 검색

6. 직원 목록 우클릭시 컨텍스트 메뉴로 라인 추가/제거가 가능합니다.

백업기능

※ 매일 최초 실행시 엑셀파일이 설치 경로에 백업됩니다. 혹시나 DB에 잘못 덮어씌웠을 경우 백업파일을 불러와주세요.

간편한 직원별 급여 계산

급여 계산

선택된 직원 또는 불러온 전체 직원에 대하여 급여 계산 및 결과물(PDF)를 출력하는 페이지입니다.

처리 순서

1. 급여계산을 실행 할 직원을 선택하면 즉시 급여가 계산됩니다.

2. 급여 계산 후 PDF출력, 메일 발송이 가능합니다.

기능

1. 직원 직급별 필터

2. 보험료 고지내역 업로드가 되었다면 고지내역에 있는 월을 급여계산월로 강제합니다.

3. 우측 하단에 각종 수당 및 혜택에 대한 기간을 표시합니다.

4. 국민연금/건강보험/장기요양보험료는 고지내역의 값을 가져옵니다.

5. 급여 계산월에 연장근무, 휴일근무가 있을 경우 상세 내역을 제공합니다.

6. PDF는 주민등록번호 앞자리가 비밀번호입니다.

주의

※ 보낸 메일은 회수가 되지 않습니다. 주의하세요.

사용자 맞춤 환경 설정

일반 설정

  • 데스크톱 알림 사용: 알림을 바탕화면에 표시합니다.
  • 자동 백업 사용: 1일 1회 최초 실행시 직원목록을 엑셀파일로 백업합니다.
  • 앱 시작 시 자동 업데이트 확인: 프로그램을 시작할 때마다 최신 버전이 있는지 확인합니다.
  • 세액표 업로드: 간이세액표가 개정되었을 경우 업로드해야합니다.

회사 설정

회사 정보, 4대 보험 요율, 급여 명세서 발송을 위한 이메일(SMTP) 서버 정보 등 급여 관리에 필요한 핵심 정보를 설정하고 관리할 수 있습니다.

  • 고용보험 요율 및 국민연금 상하한액이 개정되었을 경우 수정해야합니다.

자주 묻는 질문 (FAQ)

Q. 메일 테스트가 실패합니다.

  • 네이버웍스 등 메일 서비스의 관리자 페이지에서 SMTP 사용이 활성화되어 있는지 확인하세요.
  • 2단계 인증을 사용하는 경우, 일반 비밀번호 대신 '앱 비밀번호'를 생성하여 사용해야 합니다.
  • 사내 방화벽에서 SMTP 포트(465 또는 587)가 차단되어 있지 않은지 확인하세요.

Q. 환경 설정 내용이 저장되지 않습니다.

  • 회사 설정 탭의 내용을 변경한 후, 반드시 우측 하단의 [설정 저장] 버튼을 클릭해야 합니다.
  • 데이터베이스 연결에 문제가 없는지 확인해주세요. (일반 설정은 자동 저장됩니다.)
