Salarium에 오신 것을 환영합니다!

Salarium이란?

복잡하고 번거로운 급여 계산 및 관리 업무를 간소화하는 스마트 급여 관리 프로그램입니다.

기대 효과

단순 반복 업무에서 벗어나 더 중요한 핵심 업무에 집중할 수 있도록 돕습니다. 정확하고 빠른 급여 처리로 업무 효율을 극대화하세요.

설치 및 시작하기

시스템 요구 사항

  • 운영체제: Windows 10 이상
  • .NET 런타임: .NET 8.0 이상

설치 방법

  1. Salarium-Setup-vX.X.X.exe 파일 실행
  2. 화면 안내에 따라 설치 진행
  3. 바탕화면 아이콘으로 프로그램 시작

자동 업데이트

프로그램 시작 시 항상 최신 버전으로 자동 업데이트됩니다. 최상의 경험을 위해 인터넷 연결을 유지해주세요.

한눈에 보는 메인 화면

프로그램 메인 화면 스크린샷 영역

1. 메인 메뉴

핵심 기능으로 빠르게 이동

2. 직원 목록

관리 중인 모든 직원 확인

3. 작업 영역

실제 급여 계산 및 정보 입력

4. 상태 표시줄

중요 알림 및 현재 상태 표시

간편한 직원 정보 관리

직원 추가

직원 추가 화면 스크린샷

'직원 추가' 버튼을 클릭한 후, 이름, 입사일, 급여 정보 등 필수 항목을 입력하여 새로운 직원을 등록할 수 있습니다.

정보 수정 및 삭제

직원 목록에서 특정 직원을 선택하여 정보를 수정하거나, 더 이상 관리하지 않는 직원의 정보를 삭제할 수 있습니다.

급여 계산하기 (단계별 가이드)

1

보험료 엑셀 파일 업로드

사회보험통합징수 사이트에서 다운로드한 연금보험, 건강보험료 산출내역(CSV) 파일을 프로그램에 업로드합니다.

※ 경로: 고지내역조회 → 보험료 산출내역 조회 → 연금/건강 선택 → 산출내역출력하기

2

업로드 내역 적용

업로드된 보험료가 정확한지 확인한 후, [적용하고 직원관리로 이동] 버튼을 클릭하여 직원 정보에 반영합니다.

3

급여 계산 실행

직원관리 탭에서 급여를 계산할 직원을 선택한 후, [선택직원 급여계산] 버튼을 클릭합니다.

4

결과 확인 및 명세서 발송

계산 완료 후, 직원을 선택하여 개별 급여명세서를 출력하거나 이메일로 발송할 수 있습니다. 전체 명세서 출력 및 전체 메일 발송도 가능합니다.

클릭 한 번으로 급여 명세서 발송

발송 방법

급여 계산이 완료된 후 '명세서 발송' 버튼을 클릭하면, 등록된 직원의 이메일 주소로 급여 명세서가 자동으로 발송됩니다.

강력한 보안

모든 급여 명세서는 암호화되어 안전하게 전송되므로 민감한 개인정보를 보호할 수 있습니다.

사용자 맞춤 환경 설정

일반 설정

  • 데스크톱 알림 사용: 중요한 알림을 바탕화면에 표시합니다.
  • 자동 백업 사용: 데이터를 주기적으로 자동 백업하여 안전하게 보관합니다.
  • 앱 시작 시 자동 업데이트 확인: 프로그램을 시작할 때마다 최신 버전이 있는지 확인합니다.
  • 알림 표시 시간: 데스크톱 알림이 화면에 표시되는 시간을 초 단위로 조절합니다.

회사 설정

회사 정보, 4대 보험 요율, 급여 명세서 발송을 위한 이메일(SMTP) 서버 정보 등 급여 관리에 필요한 핵심 정보를 설정하고 관리할 수 있습니다.

자주 묻는 질문 (FAQ)

Q. 메일 테스트가 실패합니다.

  • 네이버웍스 등 메일 서비스의 관리자 페이지에서 SMTP 사용이 활성화되어 있는지 확인하세요.
  • 2단계 인증을 사용하는 경우, 일반 비밀번호 대신 '앱 비밀번호'를 생성하여 사용해야 합니다.
  • 사내 방화벽에서 SMTP 포트(465 또는 587)가 차단되어 있지 않은지 확인하세요.

Q. 환경 설정 내용이 저장되지 않습니다.

  • 회사 설정 탭의 내용을 변경한 후, 반드시 우측 하단의 [설정 저장] 버튼을 클릭해야 합니다.
  • 데이터베이스 연결에 문제가 없는지 확인해주세요. (일반 설정은 자동 저장됩니다.)
